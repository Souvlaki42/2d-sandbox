# 2D Sandbox

A 2D sandbox game like Terraria, made with Godot 4.5+.

![Preview screenshot](screenshots/preview.png)

## Features

- Terrain generation.
- Tree generation.
- Ore generation.
- Biomes.
- Player.

## Resources

- [2D Voxel Asset Pack by Kenney](https://www.kenney.nl/assets/voxel-pack)
- [Crosshair Pack by Kenney](https://www.kenney.nl/assets/crosshair-pack)

## How to run

Import the godot project file, inside Godot 4.5+.
