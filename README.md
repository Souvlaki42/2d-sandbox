# 2D Sandbox

A 2D sandbox game like Terraria, made with Godot 4.5+.

![Preview screenshot](preview.png)

## Features

- Very basic empty terrain generation.
- Very basic camera movement.

## How to run

Import the godot project file, inside Godot 4.5+.
